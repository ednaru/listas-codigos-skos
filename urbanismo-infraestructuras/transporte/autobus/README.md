En esta carpeta se guarda información sobre tesauros SKOS relacionados con el vocabulario de tranporte.

Concretamente, se guarda información sobre el SKOS Concept Scheme http://vocab.linkeddata.es/datosabiertos/kos/urbanismo-infraestructuras/transporte/autobus/tipo-equipo-asistencia, que permiten definir diferentes tipo autobús según equipo de asistencia reconocidos por el Consorcio Regional de Transportes de Madrid.
